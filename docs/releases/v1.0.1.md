# Codex Link v1.0.1

Patch release for Codex account authentication from the Flutter app.

## Highlights

- Added Codex account controls in Settings for device-code login, browser login, API-key login, logout, and refresh.
- Added host bridge protocol support for app-server account status and login events.
- Fixed the API-key dialog crash when cancelling the prompt.

## Install

Download `codex-link-v1.0.1-universal.apk` from this release for the broadest Android compatibility.

Smaller ABI-specific APKs are also attached for devices that support them.
