## Highlights

- Premium ChatGPT-style Flutter UI with compact chat chrome, sidebar sessions, workspace switching, and polished tool-call cards.
- Codex app-server adapter with streaming messages, visible commands, file reads/edits, goals, plans, approvals, skills, and native session history.
- Tunnel-first remote access with cloudflared/ngrok/manual tunnel support.
- File transfer from host to phone, image previews in chat, file mentions with `@`, and `/send` file offers.
- Settings, command center, model/effort controls, yolo toggle when host allows it, and themed accent selection.
- GitHub-release update check in the app with a Settings -> Updates download flow.

## Install

Download `codex-link-v1.0.0-arm64.apk` from this release and install it on Android arm64 devices.

This APK is signed with the current debug signing configuration, so it is intended for direct testing, not Play Store distribution.

## Update Flow

The app checks GitHub Releases for newer tags. When a newer release includes an `.apk` asset, the app shows an update notice and opens the APK download in the system browser.
