# Codex Link v1.0.2

Patch release for app-server workflow, chat controls, and desktop packaging.

## Highlights

- Added Linux desktop packaging alongside the Android release APK.
- Improved app-server workflow handling for subagent activity, tool output, and chat noise.
- Added host update support and npm host package metadata.
- Refined chat controls, settings placement, usage display, notifications, file open actions, voice input, and shell access.

## Install

Android: download `codex-link-v1.0.2-universal.apk`.

Linux: download `codex-link-v1.0.2-linux-x64.tar.gz`, extract it, then run `./bundle/codex_lan_flutter`.
